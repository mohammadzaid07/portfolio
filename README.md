## üöÄ Getting Started

### 1. Clone the Repo

```bash
git clone https://github.com/mohammadzaid07/portfolio.git
cd portfolio
```

### 2. Install Dependencies

```
npm install
```

### 3. Setup EmailJS

Create a `.env` file in the root with:

```
VITE_EMAILJS_SERVICE_ID=your_service_id
VITE_EMAILJS_TEMPLATE_ID=your_template_id
VITE_EMAILJS_API_PUBLIC_KEY=your_public_key
```

> üîë Replace with your actual credentials from [EmailJS Dashboard](https://dashboard.emailjs.com/admin/account).

### 4. Run Locally

```
npm run dev
```

### 5. Build for Production

```
npm run build
```
